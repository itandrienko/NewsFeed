# NewsFeed version 1.0

React / TypeScript / Babel / Webpack / Node

## Install

Clone the project

```bash
  git clone https://github.com/itandrienko/NewsFeed.git
```

Install dependencies

```bash
  npm install
```

Build the server

```bash
  npm run build
```
## Run

```bash
  npm run start
```
